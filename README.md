# 📇 Contact Manager

**Contact Manager** — это консольное приложение на Python для управления контактами. Проект разработан с упором на чистую архитектуру, разделение логики по модулям и работу с постоянным хранилищем данных в формате JSON.

## Возможности
 
* ➕ Добавление новых контактов
* 📋 Просмотр всех контактов
* 🔍 Поиск контактов по имени, телефону и email
* ❌ Удаление контактов по ID
* 📂 Просмотр списка избранных контактов
* ✏️ Редактирование информации о контакте
* 🔃 Сортировка контактов
* 💾 Автоматическое сохранение данных в `contacts.json`

## Используемые технологии

* Python 3
* JSON
* Модуль `json`
* Модульная архитектура

## Структура проекта

```text
Contact_Manager/
│
├── main.py          # Главное меню
├── contacts.py      # Логика работы с контактами
├── storage.py       # Загрузка и сохранение данных
├── validators.py    # Проверка пользовательского ввода
└── contacts.json    # Хранилище контактов
```
